Installer sur Android

  • 1. Appuie sur “Télécharger l’APK”.
  • 2. Ouvre le fichier téléchargé puis accepte “Autoriser cette source” si Android le demande.
  • 3. Installe. Lance l’app depuis l’écran d’accueil. Terminé.
  • 💡 Conseil — si une ancienne version est installée, désinstalle-la pour mettre à jour l’icône et éviter les conflits de signature.
  • 📲 Mises à jour — reviens sur cette page pour récupérer la nouvelle version.
  • 🔐 Sécurité — l’APK est fourni par l’éditeur du jeu. Tu peux toujours le scanner avec ton antivirus si ça te rassure.

    • Sur iPhone (iOS)

    • 1. Ouvre la version web dans Safari.
    • 2. Bouton Partager → Ajouter à l’écran d’accueil pour créer un raccourci plein écran.
    • 3. Lance le jeu depuis l’icône créée. C’est la méthode la plus simple sans App Store.
